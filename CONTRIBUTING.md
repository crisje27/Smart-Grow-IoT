# Contribuir a SMART GROW

¡Gracias por tu interés en contribuir! 🌱

## Cómo Contribuir

### Reportar Bugs
1. Verificar que no exista un issue similar
2. Crear nuevo issue con descripción detallada
3. Incluir pasos para reproducir

### Pull Requests
1. Fork el repositorio
2. Crear rama: `git checkout -b feature/nueva-funcionalidad`
3. Commits descriptivos: `git commit -m "feat: agregar sensor pH"`
4. Push: `git push origin feature/nueva-funcionalidad`
5. Crear Pull Request

### Convención de Commits
- `feat:` Nueva funcionalidad
- `fix:` Corrección de bug
- `docs:` Documentación
- `refactor:` Refactorización

## Áreas de Contribución
- Hardware: Nuevos sensores, optimización
- Software: API, frontend, ML
- Documentación: Tutoriales, traducciones

## Contacto
Issues de GitHub para bugs y features.

# Contribuir a SMART GROW

¡Gracias por tu interés en contribuir a SMART GROW! 🌱

## Código de Conducta

Este proyecto adhiere a un código de conducta amigable y respetuoso. Por favor, lee y respeta las normas de la comunidad.

## Cómo Contribuir

### Reportar Bugs

1. Verifica que el bug no haya sido reportado previamente en [Issues](https://github.com/crisje27/SMART-GROW-SISTEMA-IoT-LoRa-CON-MACHINE-LEARNING-PARA-AGRICULTURA-DE-PRECISI-N/issues)
2. Abre un nuevo Issue con:
   - Título descriptivo
   - Pasos para reproducir
   - Comportamiento esperado vs actual
   - Capturas de pantalla si aplica
   - Versión del software/hardware

### Sugerir Mejoras

1. Abre un Issue con el tag `enhancement`
2. Describe la mejora propuesta
3. Explica el caso de uso

### Pull Requests

1. Fork el repositorio
2. Crea una rama para tu feature: `git checkout -b feature/nueva-funcionalidad`
3. Haz commits descriptivos: `git commit -m "feat: agregar sensor de pH"`
4. Push a tu fork: `git push origin feature/nueva-funcionalidad`
5. Abre un Pull Request

### Convención de Commits

Usamos [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` Nueva funcionalidad
- `fix:` Corrección de bug
- `docs:` Documentación
- `style:` Formato (no afecta código)
- `refactor:` Refactorización
- `test:` Tests
- `chore:` Mantenimiento

### Estilo de Código

**Python:**
- PEP 8
- Black para formateo
- isort para imports
- Type hints

**Arduino/C++:**
- Indentación: 4 espacios
- Nombres descriptivos
- Comentarios en español

**JavaScript/Vue:**
- ESLint + Prettier
- Composición API (Vue 3)
